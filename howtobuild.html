Flutter Echo App - Build Guide

Prerequisites

Ensure you have a Linux system with snap support.

Steps

1. Clone the repository

git clone <repository-url>
cd flutter_echo

2. Install Flutter

sudo snap install flutter --classic

3. Disable analytics (optional but recommended)

flutter --disable-analytics

4. Run the app

flutter run -d linux

Notes

  • The first run will download dependencies and compile the application.
  • Press q in the terminal to quit the running app.

For other platforms (Android, iOS, macOS, Windows, Web), run flutter create --platforms=android,ios,macos,windows,web . then use flutter run -d <platform>.
